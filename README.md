# Dark Minimalist Futuristic Portfolio

Portfolio dengan tema gelap, minimalis, dan futuristik yang memiliki navigasi smooth antar halaman.

## Struktur File
- `index.html` - File HTML utama
- `styles.css` - File CSS untuk styling
- `script.js` - File JavaScript untuk interaksi
- `README.md` - Instruksi ini

## Cara Menggunakan

1. Buat folder baru untuk proyek portfolio
2. Simpan keempat file di atas dalam folder yang sama
3. Buka file `index.html` di browser

## Fitur

### 1. Tampilan Dark Theme Futuristik
- Warna gelap dengan aksen biru dan ungu
- Gradien dan efek visual futuristik

### 2. Navigasi Smooth
- Transisi halus antar halaman
- Animasi fade in/out
- Scroll otomatis ke atas saat berpindah halaman

### 3. Halaman yang Tersedia:
- **Home**: Pengenalan dengan animasi floating shapes
- **About**: Tentang saya dengan daftar skills
- **Projects**: Galeri proyek dengan card interaktif
- **Contact**: Form kontak dan informasi kontak

### 4. Responsive Design
- Optimal di desktop, tablet, dan mobile
- Menu toggle untuk mobile

### 5. Interaksi
- Hover effects pada elemen
- Form submission dengan validasi
- Background animation berdasarkan mouse movement
